## 6. 後續建議

1. **mTLS 部署**: 需在 IIS SSL Settings 設定 "Accept" Client Certificates，並在企業 CA 簽發用戶端憑證
2. **正式環境切換 RBAC**: 在 `AutofacConfig.cs` 中將 `ConfigBasedAuthProvider` 替換為 `DbRbacAuthProvider`，並建立實際 DB Schema
3. **OAuth2 外部 IdP**: 正式環境建議切換至 Azure AD / Keycloak，僅需在 `Startup.cs` 移除 OAuth Server 配置
4. **API Versioning**: 下個階段導入 Microsoft.AspNet.WebApi.Versioning
5. **Redis 快取**: 權限查詢頻繁，建議加入 Redis 快取 IAuthorizationProvider 結果
